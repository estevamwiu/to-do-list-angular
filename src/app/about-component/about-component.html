Sobre o Projeto

O To-Do-List é um projeto desenvolvido para ajudar pessoas a organizarem melhor suas tarefas e manterem o foco no que realmente importa. Ele permite adicionar, visualizar e gerenciar tarefas de forma simples e intuitiva, utilizando o poder do Angular e do framework Bulma.

Este projeto foi criado com o objetivo de aplicar conceitos de desenvolvimento web moderno, integrando boas práticas de design responsivo e uma interface limpa.
